# 个人任务看板 + 番茄钟 (Trello Pomodoro)

基于 Python FastAPI 和 Flet 的本地桌面任务管理工具，融合 Trello 看板的核心任务管理能力与番茄钟专注计时功能。

## 项目架构

```
trello-pomodoro/
├── python/
│   ├── gui.py                    # 主程序入口
│   ├── views/
│   │   ├── board_view.py         # 看板视图（含拖拽排序）
│   │   ├── card_detail.py        # 卡片详情对话框
│   │   ├── pomodoro_view.py      # 番茄钟视图（含统计弹窗）
│   │   └── stats_dialog.py       # 统计弹窗（周/月视图）
│   ├── services/
│   │   ├── board_service.py      # 看板业务逻辑
│   │   └── pomodoro_service.py   # 番茄钟业务逻辑
│   ├── storage/
│   │   └── json_storage.py       # JSON 文件读写
│   ├── models/
│   │   ├── board.py              # 看板数据模型
│   │   └── pomodoro.py           # 番茄钟数据模型
│   └── requirements.txt          # Python 依赖列表
├── data/                         # 运行时自动生成的数据目录
│   ├── boards.json               # 看板和任务数据
│   ├── pomodoro.json             # 番茄钟专注记录
│   ├── settings.json             # 用户设置
│   └── attachments/              # 上传的附件文件
└── README.md                     # 本文件
```

## 技术栈

- **GUI 层**: Python 3.9+,Flet
- **业务层**: Python 3.9+, FastAPI, Pydantic
- **通信**: HTTP REST API (本地 localhost:8765)
- **存储**: JSON 文件 (本地持久化)

## 功能模块

### 2.1 基础任务管理
- ✅ 个人看板 (创建/编辑/删除/切换背景色)
- ✅ 自定义列表 (增删改、拖拽排序)
- ✅ 任务卡片 (标题、富文本备注、拖拽切换状态)
- ✅ 子任务清单 (Checklist、进度自动更新)

### 2.2 个人时间管理
- ✅ 截止日期 (日期 + 时间、重复规则)
- ✅ 到期提醒 (15分钟/1小时/1天)
- ✅ 彩色标签 (自定义名称颜色、多标签关联)
- ✅ 任务置顶 (最多3条、图钉图标)

### 2.3 番茄钟功能
- ✅ 专注计时 (25分钟标准、开始/暂停/重置)
- ✅ 自定义时长 (5~60分钟专注、1~15分钟休息)
- ✅ 任务绑定 (卡片详情页一键启动)
- ✅ 计时提醒 (结束通知、音效)
- ✅ 专注统计 (每日/周/月统计、日历视图)
- ✅ 窗口置顶 (计时期间保持最前)

### 2.4 辅助功能
- ✅ 全局搜索 (标题、备注、子任务、标签)
- ✅ 数据归档 (已完成任务一键归档)
- ✅ 本地存储 (JSON 持久化、自动保存)

## API 接口

后端提供完整的 REST API:

| 路径 | 方法 | 说明 |
|------|------|------|
| `/api/boards` | GET/POST | 看板列表/创建 |
| `/api/boards/{id}` | GET/PATCH/DELETE | 看板详情/更新/删除 |
| `/api/lists/{id}` | POST/PATCH/DELETE | 列表操作 |
| `/api/cards/{id}` | GET/PATCH/DELETE | 卡片操作 |
| `/api/pomodoro/start` | POST | 启动番茄钟 |
| `/api/pomodoro/pause` | POST | 暂停 |
| `/api/pomodoro/stop` | POST | 停止 |
| `/api/pomodoro/stats/today` | GET | 今日统计 |

完整 API 文档见 `python/api/server.py`

**使用指南**
1.看板操作
添加列表：点击看板最右侧的“+ 添加列表”按钮。
添加卡片：点击列表底部的“+ 添加卡片”，输入标题即可。
编辑卡片：点击卡片打开详情对话框，可修改标题、备注、截止日期、提醒时间，管理附件和子任务。
移动卡片：
    跨列表移动：鼠标按住卡片拖动到另一个列表后释放，弹出对话框选择“移动到其他列表”，然后选择目标列表。
    同列表排序：拖拽卡片后选择“在当前列表排序”，使用置顶/上移/下移/置底按钮调整顺序。
删除卡片/列表：卡片右上角删除按钮（归档），列表右上角删除按钮（归档所有卡片）。

2.番茄钟使用
切换到“番茄钟”标签页。
从下拉列表选择要绑定的任务卡片（也可不选择）。
点击“开始专注”，计时开始。
计时结束后自动进入短休息/长休息（根据设置），并弹出提醒。
可随时暂停/继续/重置。
点击右上角齿轮图标自定义专注/休息时长。
点击柱状图图标查看统计（周视图/月视图）。

3.附件管理
在卡片详情对话框中，点击“上传文件”选择本地文件（支持任意格式）。
图片附件会出现“预览”按钮，点击可查看大图。
可下载或删除附件。

4.到期提醒
程序每分钟自动检查卡片截止日期。
即将到期（默认提前15分钟）或已过期的卡片会在看板中高亮显示（橙色/红色背景），并在顶部弹出通知。

5.统计视图
番茄钟界面右上角柱状图按钮打开统计对话框。
默认显示本周每日番茄数和专注分钟数。
点击“月”查看当月日历热力图（格子颜色深浅表示番茄数量）。
使用左右箭头切换不同周/月。

6.深色模式
进入程序后，点击页面右上角“深色模式”按钮，即可切换为深色模式。
切换为深色模式后，点击页面右上角“浅色模式”按钮，即可切换回浅色模式。

7.导出完成情况
进入程序后，按右上方下载图标，可以自动生成一个.txt文件（存放在reports文件夹下），其中包含具体任务完成情况。
示例如下：
【总计统计】
总专注时长:2分钟
总番茄数:2个
完成任务数:1个卡片
【按日统计】
2026-05-25
 专注时长：1分钟
 番茄数：1个
 完成任务数：0个卡片
2026-05-31
 专注时长：1分钟
 番茄数：1个
 完成任务数：1个卡片

8.今日目标设置
本功能支持设置今日专注时长目标。
在界面右上方设有设置目标和检查目标完成情况两个按钮。
设置目标按钮中，可以设置今日目标时长（单位为分钟，仅支持正整数），且支持随时修改。
检查目标完成情况按钮中，可以查看今日设置的目标、今日已经完成的专注时长，以及目标是否完成。
第二天再打开程序时，前一天设置的目标会自动重置。默认的目标为25分钟。
